#  IMDb SQL Analysis

##  Project Overview

This project analyzes an IMDb movie database using SQL. The queries explore movie information, directors, popularity, revenue, ratings, and other business insights.

---

##  Tools Used

- MySQL
- SQL
- IMDb Dataset

---

##  Database Tables

- Movies
- Directors

---

##  SQL Concepts Used

- SELECT
- WHERE
- LIKE
- ORDER BY
- GROUP BY
- COUNT
- LIMIT
- INNER JOIN

---

##  Sample Analysis

- Count total movies
- Find directors by name
- List top 3 movies by popularity
- List top 3 movies by revenue
- Find the highest-rated movie released after 2000
- Retrieve movies directed by Brenda Chapman
- Identify the director with the highest number of movies
